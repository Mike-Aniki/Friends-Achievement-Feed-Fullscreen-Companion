# Friends Achievement Feed Fullscreen Companion
![Made for Playnite Fullscreen Themes](https://img.shields.io/badge/Made%20for-Playnite%20Fullscreen%20Themes-A600FF?style=for-the-badge)

</div>

"FriendsAchievementFeed - Fullscreen Companion" is a **helper plugin for [FriendsAchievementFeed](https://github.com/justin-delano/playnite-friendsachievementfeed-plugin) in Fullscreen themes**.

It does **not** fetch achievements by itself and does **not** provide any UI.
Instead, it **reads the cache generated by the original _FriendsAchievementFeed_ plugin** and exposes the data in a **Fullscreen-friendly (theme-driven) format** via `PluginSettings`.

A compatible Fullscreen theme is required to display anything.

---

## What this plugin is (and is not)

**It is:**
- A *companion data provider* for Fullscreen themes
- A bridge between **FriendsAchievementFeed** (source plugin) and Fullscreen UI
- A lightweight loader that runs **once at Playnite startup**

**It is not:**
- A replacement for FriendsAchievementFeed
- A standalone achievement scraper
- A UI plugin

---

## Theme developers guide

All data is exposed through `PluginSettings`.
Fullscreen themes are responsible for rendering, layout, and interaction.

### Main binding

```xaml
ItemsSource="{PluginSettings Plugin=FriendsAchievementFeedFullscreenCompanion, Path=FriendGroups}"
```

This provides an accordion-ready list of groups.
Each group represents **one friend + one game**, containing a list of achievement items.

---

## Exposed data

### Root (plugin settings)

| Property | Type | Description |
|---|---|---|
| FriendGroups | ObservableCollection<FriendAchievementGroupView> | Accordion-style groups (friend + game) |
| StatusMessage | string | Error/info message (empty when OK) |
| TotalItems | int | Total number of achievements loaded (after MaxItems) |
| HasFriendsAchievements | bool | True when there is at least one achievement |
| LastRefreshLocal | DateTime? | Local timestamp of the last successful load |

> Note: this plugin runs once at startup, so *LastRefreshLocal* is the startup load time.

---

### FriendAchievementGroupView (accordion group)

| Property | Type | Description |
|---|---|---|
| EventId | string | Unique key (friend + game) |
| FriendPersonaName | string | Friend Steam display name |
| FriendSteamId | string | SteamID64 |
| FriendAvatarPath | string | Best avatar source for XAML (local path preferred) |
| FriendAvatarUrl | string | Remote avatar URL (fallback) |
| GameName | string | Game name |
| HeaderTitle | string | Preformatted title (theme can ignore and build its own) |
| HeaderSubtitle | string | Preformatted date label |
| IsExpanded | bool | Expansion state (theme-driven) |
| Items | ObservableCollection<FriendAchievementEntryView> | Achievements inside the group |

---

### FriendAchievementEntryView (achievement item)

| Property | Type | Description |
|---|---|---|
| AchievementDisplayName | string | Achievement title |
| AchievementDescription | string | Achievement description |
| FriendAchievementIcon | string | Achievement icon URL |
| UnlockTimeLocal | DateTime? | Local unlock time |
| IsRevealed | bool | Hidden/revealed state |
| FriendAvatarPath | string | Friend avatar path (auto-updated when cached avatar finishes downloading) |

---

## Avatar behavior

- Avatars are downloaded **asynchronously** (never blocks startup)
- Concurrency is limited (default: **2 downloads at once**)
- Cached locally for **7 days**
- UI is updated automatically when the local avatar becomes available

Themes should:
- Prefer `FriendAvatarPath`
- Optionally fallback to `FriendAvatarUrl`
- Or show a placeholder if empty

## XAML example (accordion UI)

Below is a real-world example of how a Fullscreen theme can render the data (header + expandable list):

```xaml
<ScrollViewer x:Name="FriendsAchievementsScroll"
              VerticalAlignment="Center"
              HorizontalAlignment="Center"
              Height="780"
              Margin="0,150,0,0"
              Background="Transparent"
              HorizontalScrollBarVisibility="Disabled"
              VerticalScrollBarVisibility="Hidden"
              CanContentScroll="True"
              FocusVisualStyle="{x:Null}"
              Focusable="False"
              IsTabStop="False"
              KeyboardNavigation.IsTabStop="False"
              KeyboardNavigation.TabNavigation="Continue"
              KeyboardNavigation.ControlTabNavigation="Continue"
              KeyboardNavigation.DirectionalNavigation="Continue">

    <ScrollViewer.Style>
        <Style TargetType="ScrollViewer">
            <Setter Property="Visibility" Value="Collapsed"/>
            <Style.Triggers>
                <DataTrigger Binding="{Binding ElementName=FriendsButton, Path=IsChecked}" Value="True">
                    <Setter Property="Visibility" Value="Visible"/>
                </DataTrigger>
            </Style.Triggers>
        </Style>
    </ScrollViewer.Style>

    <ItemsControl x:Name="FriendsAchievementsList"
                  ItemsSource="{PluginSettings Plugin=FriendsAchievementFeedFullscreenCompanion, Path=FriendGroups}"
                  Background="Transparent"
                  FocusVisualStyle="{x:Null}"
                  Focusable="False"
                  IsTabStop="False"
                  KeyboardNavigation.IsTabStop="False"
                  KeyboardNavigation.TabNavigation="Continue"
                  KeyboardNavigation.ControlTabNavigation="Continue"
                  KeyboardNavigation.DirectionalNavigation="Continue">

        <ItemsControl.ItemsPanel>
            <ItemsPanelTemplate>
                <VirtualizingStackPanel/>
            </ItemsPanelTemplate>
        </ItemsControl.ItemsPanel>

        <ItemsControl.ItemTemplate>
            <DataTemplate>
                <StackPanel Margin="0,0,0,8" Width="1600">

                    <ToggleButtonEx x:Name="HeaderToggle"
                                    Style="{DynamicResource ButtonFriendTrophy}"
                                    IsChecked="{Binding IsExpanded}"
                                    Focusable="True"
                                    IsTabStop="True"
                                    Background="Transparent"
                                    BorderThickness="0"
                                    Padding="0"
                                    Width="1580"
                                    HorizontalContentAlignment="Stretch"
                                    VerticalContentAlignment="Stretch"
                                    KeyboardNavigation.DirectionalNavigation="Continue"
                                    KeyboardNavigation.TabNavigation="Continue"
                                    KeyboardNavigation.ControlTabNavigation="Continue">

                        <Grid Height="150" Width="1580">
                            <Border x:Name="HeaderMainBorder"
                                    CornerRadius="5"
                                    Background="{DynamicResource SuccessMainBackground}"
                                    BorderThickness="3"
                                    BorderBrush="Transparent">

                                <Grid>
                                    <Grid.ColumnDefinitions>
                                        <ColumnDefinition Width="100"/>
                                        <ColumnDefinition Width="*"/>
                                        <ColumnDefinition Width="Auto"/>
                                    </Grid.ColumnDefinitions>

                                    <Grid Grid.Column="0" Margin="10,0,0,0" VerticalAlignment="Center">
                                        <Border CornerRadius="8" ClipToBounds="True">
                                            <Border Width="90" Height="90" Background="#22000000">
                                                <Image Source="{Binding FriendAvatarPath}" Width="90" Height="90"
                                                       Stretch="UniformToFill"
                                                       RenderOptions.BitmapScalingMode="Fant"/>
                                            </Border>
                                        </Border>
                                    </Grid>

                                    <StackPanel Grid.Column="1" VerticalAlignment="Center" HorizontalAlignment="Left" Margin="10,0,0,0">
                                        <TextBlock Text="{Binding HeaderTitle}"
                                                   Foreground="{DynamicResource TextColorDarkBrush}"
                                                   FontSize="32"
                                                   FontWeight="SemiBold"
                                                   TextTrimming="CharacterEllipsis"
                                                   MaxWidth="980"/>

                                        <TextBlock Text="{Binding HeaderSubtitle}"
                                                   Foreground="{DynamicResource TextColorDarkBrush}"
                                                   FontSize="22"
                                                   Opacity="0.75"
                                                   Margin="0,6,0,0"/>
                                    </StackPanel>

                                    <Grid Grid.Column="2" VerticalAlignment="Center" Margin="0,0,20,0">
                                        <Border CornerRadius="10"
                                                Background="#22000000"
                                                BorderBrush="#22FFFFFF"
                                                BorderThickness="1"
                                                Padding="10,8">
                                            <StackPanel Orientation="Horizontal">
                                                <TextBlock Foreground="{DynamicResource TextColorDarkBrush}"
                                                           FontSize="20"
                                                           Margin="0,0,5,0"
                                                           Text="{Binding Items.Count}"
                                                           FontWeight="SemiBold"/>
                                                <TextBlock x:Name="Chevron"
                                                           Text="▼"
                                                           Foreground="{DynamicResource TextColorDarkBrush}"
                                                           FontSize="22"
                                                           Opacity="0.85"/>
                                            </StackPanel>
                                        </Border>
                                    </Grid>
                                </Grid>
                            </Border>

                            <Border x:Name="HeaderSeparator"
                                    Height="1"
                                    Margin="80,0,80,-1"
                                    Background="#22FFFFFF"
                                    HorizontalAlignment="Stretch"
                                    VerticalAlignment="Bottom"/>
                        </Grid>
                    </ToggleButtonEx>

                    <ListBox x:Name="ItemsPanel"
                             ItemsSource="{Binding Items}"
                             Margin="0,10,0,0"
                             Visibility="Collapsed"
                             Background="#20000000"
                             BorderThickness="0"
                             HorizontalAlignment="Center"
                             MaxHeight="250"
                             ScrollViewer.HorizontalScrollBarVisibility="Disabled"
                             ScrollViewer.VerticalScrollBarVisibility="Auto"
                             ScrollViewer.CanContentScroll="True"
                             VirtualizingPanel.IsVirtualizing="True"
                             VirtualizingPanel.VirtualizationMode="Recycling"
                             KeyboardNavigation.DirectionalNavigation="Continue"
                             KeyboardNavigation.TabNavigation="Continue"
                             KeyboardNavigation.ControlTabNavigation="Continue"
                             SelectionMode="Single">

                        <ListBox.ItemTemplate>
                            <DataTemplate>
                                <ButtonEx HorizontalAlignment="Center"
                                          Focusable="True"
                                          IsTabStop="True"
                                          Margin="0,0,0,6"
                                          Padding="0"
                                          KeyboardNavigation.DirectionalNavigation="Continue">
                                    <Grid Width="1500" Height="110" HorizontalAlignment="Center">
                                        <Border CornerRadius="5"
                                                Background="{DynamicResource SuccessMainBackground}"
                                                BorderThickness="2"
                                                BorderBrush="Transparent"
                                                Margin="5,0,0,0">
                                            <Grid>
                                                <Grid.ColumnDefinitions>
                                                    <ColumnDefinition Width="100"/>
                                                    <ColumnDefinition Width="*"/>
                                                    <ColumnDefinition Width="Auto"/>
                                                </Grid.ColumnDefinitions>

                                                <Grid Grid.Column="0" VerticalAlignment="Center">
                                                    <Border CornerRadius="8" ClipToBounds="True">
                                                        <Border Width="80" Height="80" Background="#22000000">
                                                            <Image Source="{Binding FriendAchievementIcon}"
                                                                   Width="80" Height="80"
                                                                   Stretch="UniformToFill"
                                                                   RenderOptions.BitmapScalingMode="Fant"/>
                                                        </Border>
                                                    </Border>
                                                </Grid>

                                                <StackPanel Grid.Column="1" VerticalAlignment="Center" HorizontalAlignment="Left" Margin="10,0,0,0">
                                                    <TextBlock Text="{Binding AchievementDisplayName}"
                                                               Foreground="{DynamicResource TextColorDarkBrush}"
                                                               FontSize="30"
                                                               FontWeight="SemiBold"
                                                               TextTrimming="CharacterEllipsis"
                                                               MaxWidth="980"/>

                                                    <TextBlock Text="{Binding AchievementDescription}"
                                                               Foreground="{DynamicResource TextColorDarkBrush}"
                                                               Opacity="0.75"
                                                               FontSize="20"
                                                               Margin="0,4,0,0"
                                                               TextTrimming="CharacterEllipsis"
                                                               MaxWidth="980"/>
                                                </StackPanel>

                                                <Grid Grid.Column="2" VerticalAlignment="Center" Margin="0,0,24,0">
                                                    <TextBlock Text="{Binding UnlockTimeLocal, StringFormat='{}{0:HH:mm}'}"
                                                               Foreground="{DynamicResource TextColorDarkBrush}"
                                                               FontSize="24"
                                                               HorizontalAlignment="Right"/>
                                                </Grid>
                                            </Grid>
                                        </Border>
                                    </Grid>
                                </ButtonEx>
                            </DataTemplate>
                        </ListBox.ItemTemplate>
                    </ListBox>
                </StackPanel>

                <DataTemplate.Triggers>
                    <DataTrigger Binding="{Binding IsExpanded}" Value="True">
                        <Setter TargetName="ItemsPanel" Property="Visibility" Value="Visible"/>
                        <Setter TargetName="Chevron" Property="Text" Value="▲"/>
                    </DataTrigger>

                    <Trigger SourceName="HeaderToggle" Property="IsFocused" Value="True">
                        <Setter TargetName="HeaderSeparator" Property="Visibility" Value="Collapsed"/>
                    </Trigger>
                </DataTemplate.Triggers>
            </DataTemplate>
        </ItemsControl.ItemTemplate>
    </ItemsControl>
</ScrollViewer>
```
